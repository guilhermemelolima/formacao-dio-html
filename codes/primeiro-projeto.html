Internet como surgiu

(↑ voltar)

A internet e os computadores evoluíram de forma paralela; podemos dizer que a internet está ligada principalmente à comunicação. Um dos principais instrumentos de comunicação foi o telégrafo, que usava código morse como forma de transmitir uma mensagem. Na década de 50, no período da Guerra Fria, a disputa entre **USA** e **URSS** causou a explosão da tecnologia. Desse período surgiu a DARPA, que tinha por objetivo manter a superioridade tecnológica dos EUA. Joseph Carl Robnett Licklider "plantou" a semente na DARPA ao teorizar sobre uma rede de computadores se comunicando em diferentes localizações. Depois de alguns avanços e evoluções, surgiu a ARPANET , que era um sistema de comunicação em pacotes. A primeira conexão aconteceu em 1969, que foi uma comunicação entre a UCL e o SRI, a mensagem era **LOGIN**, mas o que chegou do outro lado foi **LO**. A ARPANET evoluiu e ganhou outros pontos (nós), e Vinton Cerf e Robert Kahn criaram o protocolo TCP/IP, o que deu mais robustez e deu origem a microredes especializadas. Em 1989, Tim Berners-Lee uniu o TCP/IP e hipertexto para desenvolver a WWW, e o primeiro site foi o "info.cern.ch".

Grandes Nomes

(↑ voltar)
  • Grace Murray Hopper: Foi uma das primeiras programadoras do Mark I, criou a linguagem de alto nível Flow-Matic (base do COBOL) e deu origem ao termo "bug".
  • Joseph Carl Robnett Licklider: Teorizou sobre uma rede de computadores e, por esta teoria, foi recrutado pela DARPA. Plantou a semente da comunicação dando a ideia de conectar dois computadores em pontos distintos.
  • Robert E. Kahn: Junto de Vinton Cerf, criou o TCP/IP e ainda apresentou o primeiro e-mail.
  • Tim Berners-Lee: Foi o inventor da WWW, diretor do WWW Consortium, popularizou o HTTP e o HTML.
  • Marc Andreessen: Reinventou o navegador com o Netscape baseado no Mosaic e tornou o navegador amigável com recursos gráficos.
  • Bill Gates: Fundador da Microsoft, a equipe da Microsoft desenvolveu o navegador Explorer que tornou a internet acessível a todos os computadores.
  • Steve Jobs: Fundador da Apple, popularizou o smartphone que conhecemos hoje.

O que são clientes

Cliente na Web

(↑ voltar)

O cliente é o dispositivo que está comunicando uma informação da internet, a qual está presente em outro computador (Servidor). O cliente vai para a internet, que busca arquivos como HTML, CSS, JavaScript, imagens, entre outros, no servidor que retorna ao cliente os arquivos que ficam armazenados na máquina na forma de cache. Os usuários utilizam uma grande quantidade de dispositivos que possuem tamanhos diferentes, o que torna importante levar em consideração o desenvolvimento de um design responsivo. Outro ponto para levar em consideração é a compatibilidade entre navegadores.

Navegadores

(↑ voltar)

Os navegadores ou browsers são softwares gratuitos que seguem os padrões W3C, mas possuem cada qual suas próprias particularidades. Tiveram início com o Mosaic, que passou para Netacape e hoje possuímos uma ampla variedade.

Aplicação web

(↑ voltar)

As aplicações são as aplicações que usam da internet como meio de comunicação ClienteXServer e não precisam de instalação. Os softwares que são instalados e os que não são instalados são diferentes, mesmo que ambos operem de forma igual; eles são de "mundos diferentes".

O que são servidores

(↑ voltar)

Servidores normalmente usam da internet para se conectar aos clientes, mas a internet não é o unico meio que poser ser usado como forma de comunicação pois eles ainda podem funcionar em rede local. Tendem a ser mais robustos, e podem ser de diferentes tipos como de arquivos, firewall, streaming, e-mail e web.

Estrutura Básica do HTML

(↑ voltar)

Um arquivo HTML é dividido em 3 partes. A primeira é a tag html, que delimita os elementos HTML (tags) e é o "primeiro nível". As outras duas partes estão no mesmo nível, mas a primeira a ser carregada pelo navegador é a head, que contém informações sobre o documento, mas não é visível aos usuários. A última parte visível para o usuário é o body.

  • <html>
    • <head>
    • </head>
    • <body>
    • </body>
  • </html>

Falando sobre tags

(↑ voltar)

Para a construção de uma página HTML, é possível encontrar uma grande quantidade de tags HTML, cada uma com sua funcionalidade específica. Algumas delas são:

  • h1 até h6: Tags usadas para títulos
  • p: Tag usada para delimitar um parágrafo
  • i: Aplica itálico a um texto
  • b ou strong: Aplica negrito a um texto
  • u: Aplica sublinhado
  • input: Gera um campo de entrada de algum tipo de valor determinado pelo atributo type
  • img: Gera uma imagem que pode ser da internet ou do computador
  • ol e ul: Tags que delimitam uma lista ordenada ou não ordenada
  • li: Identifica um item da lista
  • blockquote: Delimita uma citação em um texto
  • a: Identifica um link
  • mark: Cria uma marcação semelhante à marca texto
  • small: Diminui o tamanho de um texto
  • sub: Faz com que um determinado texto fique subscrito
  • sup: Faz com que um determinado texto fique sobescrito
  • font: É uma forma de alterar e definir uma font/texto em questão de tamanho e cor
  • del: Faz com que o texto fique com uma linha atravez dele
  • abbr: Usado em Siglas e abriviações para fornecer uma descrição com o atributo title

Atributos

(↑ voltar)
  • id: Atributo usado para a identificação dos elementos
  • style: Aplica estilo css diretamente em um atributo
  • class: liga um ou mais elementos a um estilo formatado no css
  • type: é usado junto do input para determinar o tipo de input que será usado
  • src: determina a origem de um arquivo de áudio, vídeo ou imagem
  • width: define a largura do elemento/li>
  • href: determina o destino do link
  • title: apresenta informações adicionais quando o usuário passa o mouse sobre o elemento
  • target: é usando junto de um link e diz como vai ser aberto, se será em uma nova janela "_blanck" ou guia "_"
  • size: usado com a tag font para alterar o tamanho de um texto
  • color: usado com a tag font para alterar a cor de um texto
